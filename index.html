自动化部署说明

1）GitHub Pages 负责展示静态页面；
2）GitHub Actions 每天定时抓取最新价格和波动率；
3）脚本自动计算 MA200、各项得分、综合分，并写回 data/*.json；
4）你每天打开网页，就能直接看到最新结果。

注意：默认自动抓 NASDAQ100 和 VIXCLS；PE 默认读取 data/pe_history.json，也支持用环境变量或自定义 JSON 地址覆盖。

浏览器内临时覆盖

这里的修改只在当前浏览器里生效，不会改 GitHub 上的自动数据文件。正式自动更新由 scripts/fetch_data.py 和工作流完成。
